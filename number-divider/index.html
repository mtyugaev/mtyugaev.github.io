number-divider.js

jQuery number divider
Example 1
<div class=”divide”>500</div>
Example 2
<input type=”text” class=”divide”/>

Install

Download

number-divider.js

Get in Github
Bower
bower install number-divider
Include the number-divider.js file in your site
<script src="lib/number-divider.min.js"></script>
Initialize with jQuery
$('.divide').divide();
Options
$('.divide').divide({
          
   // current delimiter
   delimiter:' ',
          
   // 1000 or 1,000
   divideThousand:false
 
});
